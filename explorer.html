📊 How to Use Data Explorer:

  1. Add a Table: Click the "+ Add Table" button above
  2. Choose Data Source: Upload a CSV file, use a URL, or select a preset
  3. Select Rows: Check the boxes next to rows you want to use for training
  4. Send to Training: Click "Send to Training" when you're ready

💡 Pro Tip: You can add multiple tables and combine data from different sources. The system will keep track of which rows came from which table.
